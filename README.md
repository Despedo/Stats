## Components
 - ESP8266
 - RGB LED (can be on ESP board)
 - OLED Display (0.96 IN, 128X64, SSD1306, I2C)

## Wiring
### Display
 - *GPIO5* (D1) -> **SCL**
 - *GPIO4* (D2) -> **SDA**
 - *GND* -> **GND**
 - *3.3v* -> **VDD**
### RGB LED
 - *GPIO12* (D6) -> **GREEN LED** 
 - *GPIO15* (D8) -> **RED LED** 
 - *GPIO13* (D7) -> **BLUE LED** 
 - *GND* -> **GND**
